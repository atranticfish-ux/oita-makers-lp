# OITA MAKERS ランディングページ

## プロジェクト概要
大分のものづくり産業の魅力を若い世代に伝えるブランディングLP。

## ファイル構成
- `index.html` — メインページ（全セクション）
- `css/style.css` — スタイルシート
- `js/main.js` — スクリプト
- `image/` — 画像ファイル

## デプロイ情報
- GitHubリポジトリ: https://github.com/atranticfish-ux/oita-makers-lp
- 公開URL: https://atranticfish-ux.github.io/oita-makers-lp/
- ホスティング: GitHub Pages（mainブランチから自動デプロイ）

## 更新手順
ファイルを編集後、以下を実行するとサイトに反映される：
```
git add .
git commit -m "更新内容のメモ"
git push
```
反映まで数分かかる場合がある。
